# TokenTune 🎶
**RFID Mood Tokens for Instant Music Playback with IPS LCD Feedback**

TokenTune is an interactive hardware-software project that uses RFID mood tokens to trigger music playback instantly. When an RFID token is placed, the system detects it, plays a specific music track, and displays feedback on an IPS LCD. Designed for rapid music selection without a screen-touch interface.

---

## 🚀 Features

🧠 **RFID‑Driven Interaction**  
Each token corresponds to a mood or music set — just place the card, and the system reacts instantly.

🎵 **Instant Music Playback**  
TokenTune connects to a music library and plays the assigned track immediately when a tag is scanned.

📺 **IPS LCD Feedback**  
Shows current status, track info, or visuals directly on a connected IPS LCD display.

🔌 **Simple Controls**  
Minimal buttons required — everything driven by token presence.
